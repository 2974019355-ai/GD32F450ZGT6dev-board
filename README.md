# GD32F450ZGT6dev-board
本项目是GD32F4开发板项目，包含开发板最小系统以及外围TF卡、SDRAM、RGB显示屏、8080并口的一个四层板项目。/This project is a GD32F4 development board design, a four-layer PCB that includes the minimum system circuit of the GD32F4 microcontroller, along with peripheral modules such as a TF card interface, SDRAM, an RGB display screen, and an 8080 parallel port.
